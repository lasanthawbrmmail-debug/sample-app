Simple Demo App

Quick notes

A tiny single-file HTML demo that you can upload to GitHub Pages. It demonstrates a small interactive UI + local storage so the site feels "real" even though it's static.

About this demo

Features included:

  • Single HTML file (no build step)
  • Persistent notes using localStorage
  • Small vanilla JavaScript for interactivity
  • Responsive layout
How to use

Type a short note and click Add. Notes are saved to your browser's local storage.

Created for a demo GitHub Pages deployment — copy this file into a repository as index.html and push.
